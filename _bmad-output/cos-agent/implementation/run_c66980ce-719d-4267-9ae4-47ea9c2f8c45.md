## Design Differentiation Notes

**Status:** pass
**Intended archetype:** chat-first inbox

- **Visible choices made:** Dark `bg-gray-950` background on all views; indigo left accent bar (`w-1 bg-indigo-600`) in ProjectInboxView; compact left sidebar with `border-r` divider in ChatView; 4-step visual progress indicator in OnboardingWizard; oversized gray 404 number in NotFoundView.
- **Layout silhouette difference from default:** ProjectInboxView has a narrow left accent bar + inbox column (not a centered card grid). ChatView placeholder has explicit 3-column structure (sidebar | chat | log slot) visible even without content.
- **Default convergence risks avoided:** No generic light-background card grid; no centered hero layout on the first screen.
- **Remaining weak spots:** Placeholder text labels are still visible in all views; full inbox thread row structure with status badges and timestamps comes in story-2.3.
