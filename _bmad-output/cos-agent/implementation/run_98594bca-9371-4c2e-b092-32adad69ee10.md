## Design Differentiation Notes

**Status:** pass
**Intended archetype:** chat-first inbox

**Visible choices made:**
- Narrow left icon rail (`w-14`, gray-900 bg) with "AS" logo pill — messaging-app nav rail pattern, not a full sidebar
- Thread rows: avatar circle (first letter of project name, indigo-tinted) + bold name + right-aligned timestamp on line 1; monospace path + colored status badge on line 2
- Hover: left indigo-500 border accent via `before:` pseudo-element — inbox active-thread selection cue
- Monospace font for file path — developer-contextual without dominating the UI
- Korean relative timestamps: "방금 전", "3시간 전", "어제" — natural language, not raw ISO dates

**Layout silhouette difference from default:**
Two-pane (icon rail + inbox) with avatar-led thread rows — structurally distinct from any centered card grid or plain list. A user would immediately recognize the messaging/inbox metaphor.

**Default convergence risks avoided:**
- No card grid layout
- No centered content area
- Thread rows have avatar + two-line metadata (not plain list items with just a title)

**Remaining weak spots:**
- Add button destinations to `/projects/new` placeholder until story-2.4 implements modal
- No unread/active indicator on rows yet (no execution data available in this story)
