# Agent Studio

CLI 환경에서 사용하기 어렵고 불편한 OpenClaw와 같은 에이전트 도구를 웹 UI로 감싸서, 사용자가 터미널 명령어를 몰라도 쉽게 사용할 수 있게 만드는 Wrapper 서비스입니다.

KiloClaw, StatClaw 같은 기존 Wrapper 서비스를 참고해, 사용자가 브라우저에서 프로젝트를 선택하고, 채팅창에 작업 요청을 입력하고, 실행 결과와 로그를 실시간으로 확인할 수 있는 웹 기반 인터페이스를 제공합니다.

핵심은 새로운 AI 에이전트 엔진을 만드는 것이 아니라, 기존 CLI 기반 에이전트 도구를 더 쉽게 실행하고 관리할 수 있도록 UI/UX를 제공하는 것입니다.
